GitHub Copilot revolutionized the way developers write code with its AI-powered autocompletion feature.

**What happened:** On June 29, 2021, GitHub announced GitHub Copilot, a code completion and programming AI-assistant developed by GitHub and OpenAI. The tool was initially released to individual developers and businesses in 2022. Currently available in various integrated development environments (IDEs), GitHub Copilot assists users with autocompleting code. [GitHub Copilot](https://en.wikipedia.org/wiki/GitHub_Copilot)

**Why it matters:** By 2023, GitHub reported that Copilot users wrote 55% of their code with AI assistance, making the question 'will AI take my job?' viscerally real for software developers. This milestone highlights the growing adoption of generative artificial intelligence in the coding process and its potential to transform the way developers work.
