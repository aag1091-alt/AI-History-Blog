# GitHub Copilot: AI Writes Code for You

On June 29, 2021, GitHub and OpenAI launched GitHub Copilot, an AI tool that writes code for developers, marking a significant milestone in the integration of AI into software development.

**What happened:** On June 29, 2021, GitHub and OpenAI announced the release of GitHub Copilot, an AI tool designed to assist developers by autocompleting code. Developed by the Codex team at OpenAI and led by Nat Friedman, GitHub Copilot leverages a large language model to suggest code snippets based on context. The tool was initially made available to individual developers and businesses on a subscription basis.

**Why it matters:** GitHub Copilot's launch was a pivotal moment, as it was the first AI tool to integrate seamlessly into the daily workflow of millions of knowledge workers. By 2023, GitHub reported that users were writing 55% of their code with AI assistance. This development raised profound questions about the future of software development and the role of AI in the workplace. As Armin Ronacher, an early user, noted, "With GitHub Copilot, I don't have to think about the syntax anymore. I just think about the problem."
